_Advantage of the PHP and language quarks_
1. Own templating engine. You don't need a special library to inject variables into HTML
2. In PHP, the server does all that heavy lifting for you before your script even runs. It takes all the incoming web data and automatically populates "Superglobals"
3. PHP uses a "Shared-Nothing" architecture. When a user requests a PHP page, the server starts up PHP, runs the script, spits out the HTML, and then completely kills the script and deletes all its memory.
